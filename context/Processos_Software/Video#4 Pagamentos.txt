# Procedimento Operacional Padrão (POP)

## 1. Objetivo

Fornecer instruções passo a passo para navegar e gerir pagamentos e dívidas de utentes no software SportStudio (v. 3.08.8811), incluindo visualização, edição, eliminação e seleção para pagamento.

## 2. Âmbito

Este POP aplica-se a todos os utilizadores responsáveis pela gestão financeira de contas de utentes nos módulos "Utentes", "Contrato", "Conta Corrente" e "Pagamentos" do SportStudio.

## 3. Pré-requisitos

- Utilizador com sessão iniciada na aplicação SportStudio.
- Acesso aos módulos "Utentes", "Contrato", "Dívidas" (Conta Corrente) e "Pagamentos".

## 4. Procedimento

1.  **[00:28]** - Clicar no botão "Voltar" (seta azul para a esquerda) no ecrã "Agenda" para navegar para o ecrã "Contrato".
2.  **[00:31]** - Clicar no botão "Voltar" no ecrã "Contrato" para navegar para o ecrã "Utentes".
3.  **[00:33]** - Certificar-se de que o utente correto (ex: "Teste Formação") está selecionado na lista de utentes.
4.  **[00:37]** - Clicar no botão "Dívidas" na barra de ferramentas para aceder ao ecrã "Conta Corrente" do utente selecionado.
5.  **[00:49]** - Clicar no botão "Voltar" para regressar ao ecrã "Utentes".
6.  **[00:52]** - Clicar no botão "Pagar" na barra de ferramentas para aceder ao ecrã "Pagamentos" do utente selecionado.
7.  **[01:01]** - Clicar no botão "Voltar" para regressar ao ecrã "Utentes".
8.  **[01:03]** - Clicar no botão "Dívidas" novamente para aceder ao ecrã "Conta Corrente".
9.  **[01:09]** - Clicar no botão "Voltar" para regressar ao ecrã "Utentes".
10. **[01:12]** - Clicar no botão "Contrato" na barra de ferramentas para aceder ao ecrã "Contrato" do utente.
