# Procedimento Operacional Padrão (POP)

## 1. Objetivo

Fornecer instruções passo a passo para atribuir um perfil de contrato dentro do software SportStudio (v. 3.08.8811).

## 2. Âmbito

Este POP aplica-se a todos os utilizadores responsáveis pela gestão de contas de utentes no módulo "Utentes" do SportStudio.

## 3. Pré-requisitos

- Utilizador com sessão iniciada na aplicação SportStudio.
- O ecrã/separador "Utentes" está aberto.
- O utente estar como cliente Individual

## 4. Procedimento

1.  Clicar no botão "Perfil".
2.  Clicar na opção de perfil "Mensalidade Humans Performance" no menu suspenso.
3.  Clicar no ícone de calendário ao lado de "Data de referência".
4.  Clicar no dia '1' no calendário (para definir a data de início como 2023-09-01).
5.  Clicar no botão "Seguinte >".
6.  Clicar na caixa de verificação da parcela "Inscrição Anual Myo Clinic".
7.  Clicar na caixa de verificação da parcela "Fisioterapia\_BM\_2x/Semana/Mensal - Base".
8.  Clicar no botão "Concluir".

## 5. Resultado Esperado

O estado da conta do utente selecionado é alterado com sucesso para 'Ativo' no sistema SportStudio, a partir da data de início escolhida. A lista de utentes reflete esta alteração com o estado 'A'.
